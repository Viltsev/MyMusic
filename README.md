# MyMusic
Music service for iOS
## Content
- Technologies
- App Capabilities

## Technologies
- MVVM Architecture
- SwiftUI
- [Combine](https://developer.apple.com/documentation/combine)
- [Swinject](https://github.com/Swinject/Swinject) - Dependency Injection
- [CoreData](https://developer.apple.com/documentation/coredata/), [UserDefaults](https://developer.apple.com/documentation/foundation/userdefaults) - Data Storage
- [Firebase](https://firebase.google.com) - Authentication
- [Moya](https://github.com/Moya/Moya) - Networking
- [NavigationStack](https://developer.apple.com/documentation/swiftui/navigationstack) - Navigation
- [SDWebImage](https://github.com/SDWebImage/SDWebImageSwiftUI) - Image loading & caching
